# Code Music

Wavetable (Vital/Serum like) + Strudel-like music generation in TypeScript.

## Init

```bash
bun init -y && bun add @xterm/xterm && bun add -d typescript @types/node eslint @eslint/js globals typescript-eslint prettier
```

## Typecheck & Lint

```bash
bun run typecheck
bun run lint
```

## Dev

Code runs in browser with WebAudio API.
